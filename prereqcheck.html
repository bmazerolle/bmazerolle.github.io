ReqCheck

One of the biggest issues I have faced as a UVic Engineering student has been planning out my degree and when I should take certain courses. The most difficult part of this process has been considering the pre-requisite courses needed for each class. UVic's website does list the prerequisites needed on each course page, however students need to go down the "hyperlink rabbithole" to get to the beginning of the long line of prerequisites.

To solve this, my friends and I created PrereqCheck.tech, a website designed to provide a visual representation of the prerequisites necessary for any course offered at UVic. Using Vue.js's tree generation capabilities and python-scraped data from the UVic website and RateMyProfessor.com, users can enter any amount of courses they want at the home page and generate a tree of all prerequisites associated with the class, along with the ratings of all professors for the class from RateMyProfessor.com. We're currently working on expanding our database to include UBC courses, and eventually, an easy format to allow for any school to utilize the website.
